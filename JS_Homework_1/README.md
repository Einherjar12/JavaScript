Домашнее задание №1 — JavaScript
Тема: prompt() и alert()

Условия:
Все данные запрашивать с помощью prompt().
Результат выводить с помощью alert().

1. Запросить у пользователя его имя и вывести сообщение:
   «Привет, имя!».

2. Запросить год рождения пользователя, посчитать,
   сколько ему лет. Текущий год указать в коде как константу.

3. Запросить длину стороны квадрата и вывести его периметр.

4. Запросить радиус окружности и вывести площадь окружности.

5. Запросить расстояние между двумя городами (км)
   и за сколько часов пользователь хочет добраться.
   Посчитать необходимую скорость.

6. Реализовать конвертор валют.
   Пользователь вводит доллары, программа переводит в евро.
   Курс валюты хранить в константе.

7. Пользователь вводит объем флешки в Гб.
   Программа должна определить, сколько файлов размером
   820 Мб помещается на флешку.

8. Пользователь вводит сумму денег в кошельке и цену шоколадки.
   Программа выводит, сколько шоколадок можно купить
   и сколько сдачи останется.

9. Запросить у пользователя трехзначное число и вывести
   его в обратном порядке. Использовать оператор %.

10. Запросить целое число и определить, четное оно или нет.
    Использовать логические операторы.
    Не использовать if и switch.
